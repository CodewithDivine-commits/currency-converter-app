# Currency Converter 
This is a simple currency converter web app built with HTML,CSS and JavaScript.

## Features 
- Convert between different  currencies
- Fetches live exchange rates using an API
- Display currency flags 
- Clean, responsive UI

## Technologies Used 
- HTML
- CSS
- JavaScript
- Exchange Rate API
